# AGENTS.md

This is the public website for Engineering Decisions.

Goal:
Create a minimal static essay/blog website.

Rules:
- Use Hugo (already globally installed).
- No CMS.
- No database.
- No React.
- No Tailwind.
- No unnecessary JavaScript.
- Markdown-only posts.
- Classic essay/blog layout.
- Fast, readable, serious.
- Prioritize clarity over design but make it likeable to the eyes.
- Do not overbuild.
- This theme (paperMod) uses YAML not TOML

if you need knowledge about:
- How to install and basic info about the theme, read: https://adityatelange.github.io/hugo-PaperMod/posts/papermod/papermod-installation/
- How to customize the theme, read: https://github.com/adityatelange/hugo-PaperMod/wiki
